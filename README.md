# Prova Técnica - Nasajon

Projeto desenvolvido em Python para processar um CSV de municípios e populações.

O programa:
- Lê o arquivo input.csv
- Consulta a API de localidades do IBGE
- Enriquecimento de dados (UF, região, código IBGE)
- Gera resultado.csv
- Calcula estatísticas
- Envia os resultados para a API de correção usando autenticação Supabase.

Observação:
Tenho conhecimento inicial em Python e utilizei apoio de IA (ChatGPT) para compreender partes da implementação, principalmente requisições HTTP e integração com APIs.
